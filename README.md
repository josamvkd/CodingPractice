🚀 Java Features Practice (Java 8 → Java 22)

This repository contains hands-on coding examples and practice programs covering Java features from Java 8 to Java 22.
It is designed for developers who want to master modern Java concepts, prepare for interviews, and strengthen real-world coding skills.

**📌 Objectives**
Understand evolution of Java from Java 8 to Java 22
Practice core and advanced features with examples
Prepare for technical interviews (mid → senior level)
Build real-world coding confidence

**🧠 Topics Covered**
**🔹 Java 8 (Foundation of Modern Java)**
Lambda Expressions
Functional Interfaces
Stream API
Method References
Optional
Date & Time API

**🔹 Java 9 – 11 (Modular & API Enhancements)**
Java Module System (JPMS)
var keyword (Local Variable Type Inference)
New String Methods (isBlank(), lines(), etc.)
Files API improvements
HTTP Client API (Java 11)

**🔹 Java 12 – 17 (Language Enhancements)**
Switch Expressions
Text Blocks (Multiline Strings)
Records (Immutable Data Classes)
Pattern Matching (instanceof)
Sealed Classes
Helpful NullPointerExceptions

**🔹 Java 18 – 21 (Modern Concurrency & Performance)**
Simple Web Server
Record Patterns
Pattern Matching for switch
Virtual Threads (Project Loom)
Structured Concurrency
Scoped Values

**🔹 Java 22 (Latest Features)**
Unnamed Variables & Patterns
String Templates (Preview)
Foreign Function & Memory API
Enhanced Pattern Matching
